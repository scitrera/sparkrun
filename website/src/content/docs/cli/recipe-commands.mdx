## List recipes

```bash
sparkrun recipe list [query]
sparkrun list [query]          # alias
```

List available recipes from all configured registries. Optionally filter by a search query.

### Options

| Option | Description |
|---|---|
| `--all` / `-a` | Include recipes from hidden registries |
| `--runtime` | Filter by runtime (e.g., `vllm`, `sglang`) |
| `--registry` | Filter by registry name |

The output table includes:

| Column | Description |
|---|---|
| Name | Recipe display name |
| Runtime | Inference runtime (vllm, sglang, llama-cpp, eugr-vllm) |
| TP | Default tensor parallelism setting (`-` if not set) |
| Nodes | Minimum node count (`min_nodes`) |
| GPU Mem | Default `gpu_memory_utilization` setting (`-` if not set) |
| Registry | Source registry name or `local` |
| File | Recipe filename stem |

## Show recipe details

```bash
sparkrun recipe show <recipe>
sparkrun show <recipe>         # alias
```

Display detailed recipe information including defaults, metadata, and VRAM estimation. The VRAM estimator auto-detects model architecture from HuggingFace.

### Options

| Option | Description |
|---|---|
| `--save <path>` | Save the recipe to the specified file path |
| `--no-vram` | Skip VRAM estimation display |
| `--tp` / `--tensor-parallel` | Override tensor parallelism for VRAM estimate |

## Search recipes

```bash
sparkrun recipe search <query>
sparkrun search <query>        # alias
```

Search for recipes by name, model, or description across all registries.

### Options

| Option | Description |
|---|---|
| `--all` / `-a` | Include recipes from hidden registries |
| `--runtime` | Filter by runtime (e.g., `vllm`, `sglang`) |
| `--registry` | Filter by registry name |

The output table includes:

| Column | Description |
|---|---|
| Name | Recipe display name |
| Runtime | Inference runtime |
| TP | Default tensor parallelism setting (`-` if not set) |
| Nodes | Minimum node count |
| GPU Mem | Default `gpu_memory_utilization` setting (`-` if not set) |
| Model | HuggingFace model identifier |
| Registry | Source registry name |

## Validate a recipe

```bash
sparkrun recipe validate <recipe>
```

Check a recipe file for issues:
- Missing required fields (`model`, `runtime`)
- Invalid `mode` values
- `min_nodes` / `max_nodes` consistency
- Valid `metadata.model_params` and `metadata.model_dtype`
- Runtime-specific validation

## Estimate VRAM usage

```bash
sparkrun recipe vram <recipe> [options]
```

| Option | Description |
|---|---|
| `--tp` / `--tensor-parallel` | Override tensor parallelism |
| `--max-model-len` | Override max sequence length |
| `--gpu-mem` | Override gpu_memory_utilization (0.0-1.0) |
| `--no-auto-detect` | Skip HuggingFace model auto-detection |
