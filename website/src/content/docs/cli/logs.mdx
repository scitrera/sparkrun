---
title: sparkrun logs
description: Re-attach to inference workload logs.
---

```bash
sparkrun logs <recipe> [options]
```

Re-attach to the log output of a running inference workload. Ctrl+C safely detaches â€” it never kills the inference job.

## Options

| Option | Description |
|---|---|
| `--hosts` / `-H` | Comma-separated host list |
| `--hosts-file` | File with hosts |
| `--cluster` | Use a saved cluster by name |
| `--tp` / `--tensor-parallel` | Match host trimming from run |
| `--tail` | Number of existing log lines to show (default: 100) |

## Examples

```bash
sparkrun logs nemotron3-nano-30b-nvfp4-vllm --cluster mylab
sparkrun logs qwen3-1.7b-vllm --tp 2 --tail 50
```
