---
title: sparkrun status
description: Check the status of running inference workloads.
---

```bash
sparkrun status [options]
```

Show sparkrun containers running on cluster hosts. Displays containers grouped by job with their role, host, status, image, and ready-to-use `logs` and `stop` commands.

:::note
`sparkrun status` is an alias for the underlying `sparkrun cluster status` command.
:::

## Options

| Option | Description |
|---|---|
| `--hosts` / `-H` | Comma-separated host list |
| `--cluster` | Use a saved cluster by name |

If no options are given, uses the default cluster.

## Examples

```bash
# Check default cluster
sparkrun status

# Check a specific cluster
sparkrun status --cluster mylab

# Check specific hosts
sparkrun status --hosts 192.168.11.13,192.168.11.14
```
